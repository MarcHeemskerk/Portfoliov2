Tools of Choice

Unity

I always liked experimenting with JavaScript and other coding programs. It feels like puzzles I need to solve. I really like how I can combine my art and coding skills in Unity. During my minor Applied Game Design, I dived deeper into learning Unity and C#. Now, this is my favorite software combination which I'm planning to use more from now on. I already got lots of creative ideas for the future!

Sketch

I worked with Sketch since I did my internship at ZERO72 in 2015. That was the time when sketch was still in beta. I love how this tool helps me to bring my imagination to the digital world.

Coding

The first time I tried coding was back in 2013. With friends I owned a modded Modern Warfare 3 server. We wrote our own mods, but honestly I didn't know what I was doing at the time. In the first year (2017) of my current study, I started with HTML and CSS. After a year of HTML and CSS I discovered the power of JavaScript and right now I work a lot with C# for creating Unity Games and building applications. I also do some fun little projects for my Arduino.

Adobe

The Adobe Suite is something I'm familiar with, it feels like I grew up with it. With Photoshop I really loved the idea of being able to create and transform any image. I love to make things in After Effects, like fun little animations and putting things together. In Illustrator I can make vector illustrations, but for most web/app assets I use Sketch. If I need to make professional documents I use InDesign. All these skills I put to the test with my previous internship at NoProtocol.

Blender

Blender is a complicated but really fun tool to use. I had a few lessons in Blender during my Minor Applied Game Design and in my free time I like to experiment with it. I'm a big fan of low poly art and game assets.
